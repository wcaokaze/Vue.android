open fun <T> watcher(watchedReactiveField: ReactiveField<T>, immediate: Boolean = false, watcher: (T) -> Unit): Unit

observes a ReactiveField.

watcher invokes removeObserver automatically. You don't have to manage the Component's lifetime.

Parameters
