ComponentReactivatee

typealias ComponentReactivatee<T> = ComponentReactivateeScope.() -> T

Reactivatee which can also be reactivated by VComponentInterface.ComponentVBinder.

In other words, ComponentVBinder notifies only ComponentReactivatee, doesn't notify normal Reactivatee. ReactiveField notifies both Reactivatee and ComponentReactivatee.
